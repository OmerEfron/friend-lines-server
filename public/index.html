üîê Authentication

POST /api/auth/login
Authenticate a user and receive a JWT token.

Example Request:

curl -X POST "https://friend-lines-server.onrender.com/api/auth/login" \ -H "Content-Type: application/json" \ -d '{ "username": "yourusername", "password": "yourpassword" }'

Response 200 OK

{ "success": true, "data": { "user": { "username": "yourusername", "fullName": "Your Name", "email": "your@email.com", "uuid": "user-uuid-here" }, "token": "jwt-token-here" } }

üë• Users

POST /api/users/register
Register a new user account.

Example Request:

curl -X POST "https://friend-lines-server.onrender.com/api/users/register" \ -H "Content-Type: application/json" \ -d '{ "username": "newuser", "fullName": "New User", "email": "newuser@example.com", "password": "securepassword123" }'

Response 200 OK

{ "success": true, "data": { "user": { "username": "newuser", "fullName": "New User", "email": "newuser@example.com", "uuid": "generated-uuid", "createdAt": "2025-08-27T18:53:10.654Z" } } }
GET /api/users/profile
Get current user profile information. Requires authentication.

Example Request:

curl -X GET "https://friend-lines-server.onrender.com/api/users/profile" \ -H "Authorization: Bearer YOUR_JWT_TOKEN"

Response 200 OK

{ "success": true, "data": { "user": { "uuid": "user-uuid-here", "username": "john_doe", "fullName": "John Doe", "email": "john.doe@example.com", "createdAt": "2025-08-27T18:53:10.654Z", "updatedAt": "2025-08-27T18:53:10.654Z" } } }
GET /api/users/search?q=searchterm&page=1&limit=20
Search for users by username, full name, or email. Requires authentication.

Example Request:

curl -X GET "https://friend-lines-server.onrender.com/api/users/search?q=john&page=1&limit=10" \ -H "Authorization: Bearer YOUR_JWT_TOKEN"

Response 200 OK

{ "success": true, "data": { "users": [ { "username": "john_doe", "fullName": "John Doe", "email": "john@example.com", "uuid": "user-uuid" } ], "pagination": { "page": 1, "limit": 10, "total": 1, "totalPages": 1 } } }

ü§ù Friendships

POST /api/friendships/request
Send a friend request to another user. Requires authentication.

Example Request:

curl -X POST "https://friend-lines-server.onrender.com/api/friendships/request" \ -H "Authorization: Bearer YOUR_JWT_TOKEN" \ -H "Content-Type: application/json" \ -d '{ "friendId": "target-user-uuid" }'

Response 200 OK

{ "success": true, "data": { "success": true, "message": "Friend request sent" } }
POST /api/friendships/accept
Accept a pending friend request. Requires authentication.

Example Request:

curl -X POST "https://friend-lines-server.onrender.com/api/friendships/accept" \ -H "Authorization: Bearer YOUR_JWT_TOKEN" \ -H "Content-Type: application/json" \ -d '{ "friendId": "sender-user-uuid" }'
GET /api/friendships/list
Get list of current friends. Requires authentication.

Example Request:

curl -X GET "https://friend-lines-server.onrender.com/api/friendships/list" \ -H "Authorization: Bearer YOUR_JWT_TOKEN"
GET /api/friendships/requests
Get pending friend requests. Requires authentication.

üë• Groups

POST /api/groups/create
Create a new group. Requires authentication.

Example Request:

curl -X POST "https://friend-lines-server.onrender.com/api/groups/create" \ -H "Authorization: Bearer YOUR_JWT_TOKEN" \ -H "Content-Type: application/json" \ -d '{ "name": "My Group", "description": "A group for my friends" }'
POST /api/groups/invite
Invite a user to a group. Requires authentication and group ownership.

Example Request:

curl -X POST "https://friend-lines-server.onrender.com/api/groups/invite" \ -H "Authorization: Bearer YOUR_JWT_TOKEN" \ -H "Content-Type: application/json" \ -d '{ "groupId": "group-uuid", "invitedUserId": "user-uuid" }'
GET /api/groups/my-groups
Get groups where the user is a member. Requires authentication.
GET /api/groups/invitations
Get pending group invitations. Requires authentication.

üì∞ Newsflashes

POST /api/newsflashes/create
Create a new newsflash. Requires authentication.

Example Request:

curl -X POST "https://friend-lines-server.onrender.com/api/newsflashes/create" \ -H "Authorization: Bearer YOUR_JWT_TOKEN" \ -H "Content-Type: application/json" \ -d '{ "content": "Hello friends! This is my newsflash.", "targetType": "friends" }'

Response 200 OK

{ "success": true, "data": { "newsflash": { "authorId": "your-uuid", "content": "Hello friends! This is my newsflash.", "targetType": "friends", "createdAt": "2025-08-27T18:53:10.654Z" } } }
GET /api/newsflashes/my-feed
Get newsflashes for the authenticated user's feed. Requires authentication.

Example Request:

curl -X GET "https://friend-lines-server.onrender.com/api/newsflashes/my-feed?page=1&limit=20" \ -H "Authorization: Bearer YOUR_JWT_TOKEN"
GET /api/newsflashes/group/:groupId
Get newsflashes for a specific group. Requires authentication and group membership.
